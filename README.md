# 🔐 STM32F407VET6 智能门锁系统

基于 **STM32CubeMX + HAL库** 的完整智能门锁解决方案，具备密码验证、温湿度监测、历史记录查询、步进电机控制等功能。项目采用**状态机架构**设计，结合硬件PCB设计与完整技术文档。

## 🎯 项目概述

这是一个智能门锁控制系统的完整实现，具备以下核心功能：
- **🔐 密码验证系统**：支持密码验证、修改、管理（防暴力破解机制）
- **🔧 物理控制**：步进电机驱动（90°旋转），实现机械锁舌控制
- **📊 环境监测**：DHT11温湿度传感器实时监测
- **🖥️ 人机交互**：4×4矩阵键盘 + OLED显示界面
- **⏰ 时间记录**：开锁历史记录（时间戳存储与查询）
- **🔗 串口通信**：USART1通信接口，支持远程监控与控制
- **🔄 状态机设计**：6种状态，清晰的任务调度逻辑

## 🛠️ 硬件平台

### 核心配置
| 项目 | 规格 | 说明 |
|------|------|------|
| **MCU** | STM32F407VET6 | ARM Cortex-M4 @168MHz |

| **定时器** | TIM2/TIM3 | 定时采样与状态刷新 | 内部定时 |

## 🚀 核心功能详解

### 1. 🔐 密码管理系统 (`Password.c`)
- **密码验证**：6位数字密码验证
- **密码修改**：支持管理员密码修改
- **防暴力破解**：错误次数计数与锁定机制
- **历史记录**：开锁时间存储与查询
- **内存管理**：Flash存储密码与历史记录

### 2. 🔄 状态机设计 (`TaskDriver.c`)
采用清晰的6状态状态机，逻辑严谨：
```c
typedef enum {
    STATE_IDLE = 0,      // 空闲状态，显示主菜单
    STATE_VERTIFY,       // 密码验证状态
    STATE_OPEN,          // 已解锁状态
    STATE_CHAPWD,        // 修改密码状态
    STATE_VIEW,          // 查看历史记录
    STATE_SEL            // 管理员选择模式
} State_t;
```

### 3. 🖥️ 用户界面系统 (`Display.c`)
- **主菜单界面**：锁状态、频率、温湿度显示
- **密码输入界面**：实时显示输入字符
- **验证结果界面**：成功/失败提示
- **历史记录界面**：分页查看开锁记录
- **管理员界面**：密码修改与模式选择

### 4. ⚙️ 硬件控制
- **步进电机控制**：精确的90°旋转控制
- **矩阵键盘扫描**：硬件消抖 + 软件滤波
- **OLED显示**：IIC通信，支持中文显示
- **温湿度监测**：定时采样，超标报警

### 5. 📡 通信接口
- **串口调试**：实时输出状态信息
- **命令解析**：支持远程控制指令
- **蓝牙扩展**：预留蓝牙模块接口

## 🔧 快速开始

### 环境要求
- **开发工具**：Keil MDK-ARM v5
- **调试工具**：DAP-Link/ST-Link V2

### 编译与下载
1. **克隆项目**
   ```bash
   git clone https://github.com/Nahida-Robin/The-intelligent-storage-system-based-on-STM32.git
   ```

2. **打开工程**
   - 进入 `MDK-ARM/` 目录，双击 `.uvprojx` 文件

3. **硬件连接**
   ```
   OLED SDA → PB8
   OLED SCL → PB9
   矩阵键盘行 → PD4-PD7（上拉输入）
   矩阵键盘列 → PD0-PD3（输出）
   步进电机 → PB12-PB15
   串口调试 → PA9/TX, PA10/RX
   ```

4. **编译下载**
   - Keil中按 F7 编译
   - 连接调试器，按 F8 下载

5. **操作流程**
   ```
   上电 → 显示主菜单 → 按[11]输入密码 → 验证成功 → 解锁
   → 按[12]锁定 → 按[13]管理员模式 → 按[14]修改密码 → 按[15]查看历史
   ```
